An application to manage tasks hierarchically. 
You can create main tasks with subtasks nested within them, assign them categories (Work, Personal, Urgent, Home), due dates and receive notifications. It includes real-time search and filters by status and category. The app automatically tracks your progress and everything is saved in your browser.
